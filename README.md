# testTaskRepo
Репозиторий, куда собраны наиболее интересные мне задачи. Цель: переиспользовать данный материал для подготовки intern\junior специалистов имея актуальные задачи и решения на них, также самостоятельно вспомнить аспекты языка.

### Краткая идеология данного репозитория
В репозитории содержатся задачи уровня junior\middle, задачи уровня senior, относятся к архитектурных подходам, которые будут включены отдельными категориями
_____________
Исходный материал сгенерирован моделями Deepseek-v3.2\\gpt4.5\\glm4.7, ревью проводилось этими же моделями, где подход был применим, сгенерированная задача моделью А, проверялась моделью Б в случайном порядке.
_____________
В данном репозитории могут быть неидеальные решения, поскольку был наложен авторский (мой) отпечаток и некрасивые или непонятные решения нейросети, были заменены более простыми и удобочитаемыми вариантами.
